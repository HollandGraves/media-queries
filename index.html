the CSS that made the animation take place are:
"animation-duration" which controls the time of the animation
"animation-fill-mode" which, I presume, controls the way the animation fits over it's respective area
"0% {opacity: 1;}" which controls the base state of the animation's opacity i.e. sets the base opactiy at 100% visible
"100% {opacity: 0;}" which controls the final state of the animation's opacity i.e. sets the final opacity at 0% visible
